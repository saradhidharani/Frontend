we can used 3 types of colours in css
RGB are primary colours

1)named colours
2)rgb colours
    rgb(RED GREEN BLUE[/alpha])
    each colour is numbered from 0-255 (or) percentage 0-100%
    alpha is numbered as 0-1 (or) in terms of percentage 0-100%
3)hexadecimal colours
    #RRGGBB[AA]
    alpha value is for intensity of the colour
